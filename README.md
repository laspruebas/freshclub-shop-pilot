# Fresh Club Shop Pilot

Frontend mínimo para pedido externo por WhatsApp.

## Objetivo
Abrir una URL pública desde Landbot/WhatsApp con `household_id` por querystring, mostrar un catálogo simple y crear una orden en el backend.

## URL esperada
```text
https://TU-VERCEL-APP.vercel.app/?household_id=UUID

## Session access

The frontend is accessed through a signed session link generated by the backend.

Flow:

Landbot → backend → session link → frontend

Example:

/?token=<session_token>

The token contains the household context and is verified by the backend before allowing operations such as catalog retrieval and order creation.

The flow is:

WhatsApp
↓
Landbot
↓
backend
↓
signed session link
↓
frontend
